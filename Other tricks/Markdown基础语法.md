### 📝 Markdown 基础语法速查表

|**语法名称**|**标记符号 / 示例**|**最终效果**|
|---|---|---|
|**一级标题**|`# 标题内容`|最大的标题（建议用于文档标题）|
|**二级标题**|`## 标题内容`|次级标题（建议用于大章节）|
|**三级标题**|`### 标题内容`|常用于小节或题目名称|
|**加粗**|`**文字**`|**文字**（用于强调重点）|
|**斜体**|`*文字*`|_文字_（用于术语或补充）|
|**删除线**|`~~文字~~`|~~文字~~（用于标注废弃思路）|
|**无序列表**|`- 项目` 或 `* 项目`|• 项目（用于列举要点）|
|**有序列表**|`1. 项目`|1. 项目（用于标注步骤）|
|**待办清单**|`- [ ] 未完成` / `- [x] 已完成`|用于跟踪刷题进度|
|**行内代码**|`代码`|`代码`（用于句子中的变量或指令）|
|**代码块**|` ` `语言(如cpp/py)\n代码\n` ` `|带语法高亮的整段代码块|
|**超链接**|`[显示文字](链接地址)`|[显示文字](https://www.google.com/search?q=%E9%93%BE%E6%8E%A5%E5%9C%B0%E5%9D%80)|
|**引用**|`> 引用文字`|用于摘录题目原文或名言|
|**水平分割线**|`---`|产生一条横线（用于区分板块）|
|**表格**|`|标题|
